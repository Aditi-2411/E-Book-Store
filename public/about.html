About Our Website

Welcome to Our E-Book Lib-Store

This project is an eBook Library web application featuring user authentication, category-based book browsing, and a shopping cart system. Users can view and manage books, make payments, track orders, and maintain a wishlist. The interface is dynamic, using JavaScript for interactivity and MongoDB for data storage..

Key Features

  • User registration and login for secure access.
  • Dynamic book categories including Engineering, Novels, Horror, Fiction, and Science.
  • Interactive book management with options to rent, buy, or read books.
  • Cart system for managing selected items and proceeding with payments.
  • Wishlist feature to save favorite books for later purchase.
  • Order tracking page with dynamically generated tracking ID.
  • Quote page displaying random genre-related quotes on each refresh.
  • Document upload feature to enable PDF downloads after file upload.
  • Integrated payment flow with a confirmation process before order completion.
  • Consistent sidebar navigation across pages for a seamless user experience.

Our Vision

Our vision is to create an inclusive digital library platform that empowers users to explore, learn, and share knowledge effortlessly. We aim to enhance the reading experience by integrating modern technology with a user-friendly interface, making access to books and resources seamless for everyone. Through innovative features and community engagement, we aspire to foster a lifelong love for reading and learning.
